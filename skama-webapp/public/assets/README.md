# SKAMA Jewelry Assets

This folder contains frontend-only prototype assets.

- `css`: design tokens, themes, base rules, components, and utilities.
- `js`: browser-only interaction modules for simulated UI behavior.
- `images`: future product, collection, and editorial imagery.
- `icons`: future custom icon assets.
- `fonts`: future brand font files.
- `themes`: future theme-specific overrides.
- `utilities`: future generated or vendor-neutral frontend helpers.
